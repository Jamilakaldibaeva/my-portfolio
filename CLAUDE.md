This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## About This Project
Single-page portfolio for a **designer** targeting **freelance and contract clients**. Built with plain HTML and CSS — no frameworks, no build tools, no dependencies.

## Aesthetic
Minimal, dark.

## Rules
- Ask before restructuring anything.
- Only build what is explicitly requested — no extra features, sections, or unsolicited improvements.
- No frameworks (no Tailwind, Bootstrap, React, etc.).
- No JavaScript unless specifically asked for.
- Keep HTML semantic and CSS clean and readable.
